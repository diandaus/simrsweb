# Folder Icons

Letakkan file logo Anda di sini:

## File yang diperlukan:
- **icon.png** - Logo utama aplikasi (digunakan di sidebar)
  - Ukuran rekomendasi: 200x200px atau 512x512px
  - Format: PNG dengan background transparan
  - Ukuran file: < 500KB

## Struktur folder:
```
public/
  icons/
    icon.png          <- Logo utama aplikasi
    logo-small.png    <- Logo kecil (optional)
    logo-large.png    <- Logo besar (optional)
```

## Cara menggunakan:
1. Siapkan file icon.png Anda
2. Copy file ke folder ini (public/icons/)
3. Logo akan otomatis muncul di sidebar aplikasi
